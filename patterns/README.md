# LOOPS
There are 4 types of loops in java
   1) while Loop
   2) do-while Loop
   3) for Loop
   4) for-each loop


## while Loop
   The while loop runs many number of times based upon the condition output

```java
    Syntax
        while(condition){
        (Statment)
        }

    Example
        int i = 1;
        while(i<=100){
            System.out.println(i);
            i++;
        }
```
